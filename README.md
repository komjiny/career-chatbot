# 🎤 진로 인터뷰 챗봇 — 피지컬 AI 직업 탐색

> 고등학생이 전기전자·로봇공학 현직자와 1:1 인터뷰를 나누는 진로 탐색 앱

[![GitHub Pages](https://img.shields.io/badge/배포-GitHub%20Pages-blue?logo=github)](https://komjiny.github.io/career-chatbot)

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| **현직자 3인 선택** | 로봇제어 엔지니어 · 임베디드 개발자 · IoT 엔지니어 |
| **1:1 AI 채팅 인터뷰** | Claude AI가 현직자 페르소나로 솔직한 답변 |
| **빠른 질문 6종** | 연봉·하루일과·준비방법 등 버튼 클릭으로 즉시 질문 |
| **최신 뉴스 연동** | 대화 중 2026년 로봇·AI 뉴스 카드 자동 삽입 |
| **진로 리포트 생성** | 역량 체크리스트·추천 전공·커리어 로드맵 자동 생성 |
| **PDF 저장** | 브라우저 인쇄 기능으로 PDF로 저장 가능 |
| **데모 모드** | API 키 없이도 미리 준비된 답변으로 즉시 사용 가능 |

## 🚀 사용 방법

1. 직업 카드 선택
2. **인터뷰 시작하기** 클릭
3. 자유롭게 질문 (직접 입력 또는 빠른 질문 버튼)
4. 5회 이상 대화 후 **진로 리포트 생성** 클릭
5. 🖨️ 인쇄/PDF 저장으로 보관

---

## 🔑 API 키 설정 (선택)

API 키 없이도 데모 모드로 6가지 주제에 즉시 답변을 받을 수 있습니다.

더 풍부한 AI 답변을 원하면:
1. [console.anthropic.com](https://console.anthropic.com) 에서 API 키 발급
2. 앱 우측 상단 **API 키 설정** 클릭 후 입력
